SOUND EFFECTS — generated royalty-free by tools/generate_sfx.py (re-run anytime).

These are synthesized with ffmpeg, so there is zero licensing/copyright risk.
The editor's planner references them by name (whoosh / riser / impact / ding /
pop / swoosh) and writes timed cues into output/clip_<id>.notes.md for the
finishing pass.

Want a bigger / more realistic library? Free, no-attribution sources (2026):
  - Pixabay Sound Effects   https://pixabay.com/sound-effects/
  - Mixkit                  https://mixkit.co/free-sound-effects/
  - Zapsplat                https://www.zapsplat.com/   (free w/ account)
Drop extra .wav/.mp3 files here named for their cue (e.g. impact_2.wav).
